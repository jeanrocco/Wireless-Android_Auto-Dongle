# Wireless Android-Auto Dongle
A DIY Wireless Android Auto adapter to use with a car that supports wired Android Auto using a Raspberry Pi Zero 2W rev1.0 .


So in order not to forget about this link, here is a mere archive of https://github.com/nisargjhaveri/WirelessAndroidAutoDongle excellent and working project.

It works very well on my Honda Accord 2022. It works flawlessly with my Samsung S25. This is not the case with this 30$ https://www.amazon.ca/dp/B0FHQR79V1?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1 dongle that runs very hot, keeps restarting every 5 minutes or so, and sometimes displays a washed-up or blurry map image on my iPhone. Setting it up for my Samsung S25 was also a pain in the neck !

I used https://github.com/nisargjhaveri/WirelessAndroidAutoDongle/releases/download/v0.5.0/raspberrypizero2w-sdcard.img.xz and Raspberry Pi Imager v2.0.7 (https://downloads.raspberrypi.com/imager/imager_latest.dmg).
