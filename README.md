# Wireless Android-Auto Dongle
A DIY Wireless Android Auto adapter to use with a car that supports wired Android Auto using a Raspberry Pi Zero 2W rev1.0 ..


a mere archive of https://github.com/nisargjhaveri/WirelessAndroidAutoDongle excellent and working project.

It works very well on my Honda Accord 2022. 

I used https://github.com/nisargjhaveri/WirelessAndroidAutoDongle/releases/download/v0.5.0/raspberrypizero2w-sdcard.img.xz and Raspberry Pi Imager v2.0.7 .
